Game

Definition

Bomberman is a series of video games from Hudson Soft where the player plays a bomber, the goal is to detonate opponents / enemies to win.

The game was a great success, especially thanks to its multiplayer mode which, depending on the machines, allows to play up to ten people at the same time.
